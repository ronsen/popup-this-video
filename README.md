# popup this video

Just right click any video you want to watch and a new small window will pop up.

Supported websites
- YouTube
- Facebook
- Vimeo
- Twitch
- Metacafe
- Vidio
- DailyMotion
- gfycat
- OpenLoad

Install this extension on your Firefox browser: https://addons.mozilla.org/en-US/firefox/addon/popup-this-video/
